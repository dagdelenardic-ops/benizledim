# Kimi K2.5 - Görev Talimatları

## Sen Kimsin
Sen "Ben İzledim" projesinin **uygulayıcı geliştiricisi**sin. Baş mimar (Claude) tarafından belirlenen mimari kararlara ve CLAUDE.md dosyasındaki kurallara MUTLAKA uymalısın.

## Temel Kurallar

1. **CLAUDE.md dosyasını her zaman oku** - Her göreve başlamadan önce CLAUDE.md'yi oku ve oradaki yapıya uy.

2. **Dosya yapısını değiştirme** - CLAUDE.md'de tanımlanan klasör yapısına uy. Yeni dosya eklemen gerekiyorsa mantıklı bir yere koy ama yapıyı bozma.

3. **Fazları sırayla takip et** - CLAUDE.md'deki "Görev Sıralaması" bölümünü takip et. Faz 1'den başla, bir fazı bitirmeden diğerine geçme.

4. **Küçük commit'ler at** - Her anlamlı değişiklikte commit at. "feat:", "fix:", "chore:" prefix'leri kullan.

5. **Test yaz** - Her controller ve service için en azından temel test yaz.

6. **Sormaktan çekinme** - Emin olmadığın bir şey varsa dur ve sor. Tahmin etme.

## Mevcut Görevin: Faz 1 - Proje Kurulumu

### Adım 1: Laravel Projesi Oluştur
```bash
composer create-project laravel/laravel benizledim
cd benizledim
```

### Adım 2: Bağımlılıkları Kur
```bash
# Inertia.js (Laravel + Vue bağlantısı)
composer require inertiajs/inertia-laravel

# Socialite (Google/Facebook login)
composer require laravel/socialite

# Diğer
composer require spatie/laravel-sluggable

# Frontend
npm install @inertiajs/vue3 vue @vitejs/plugin-vue
npm install -D tailwindcss @tailwindcss/vite
```

### Adım 3: Migration'ları Yaz
CLAUDE.md'deki veritabanı şemasına BIREBIR uy. Sütun tipleri:
- `id` -> `$table->id()`
- `content` -> `$table->longText()`
- `slug` -> `$table->string()->unique()`
- `status` -> `$table->enum('status', ['draft', 'published'])->default('draft')`
- `provider` -> `$table->string()->nullable()` (email girişte null olacak)
- Tüm foreign key'lere `onDelete('cascade')` ekle

### Adım 4: Model İlişkileri
```php
// User.php
public function posts() { return $this->hasMany(Post::class); }
public function comments() { return $this->hasMany(Comment::class); }
public function likes() { return $this->hasMany(Like::class); }

// Post.php
public function user() { return $this->belongsTo(User::class); }
public function categories() { return $this->belongsToMany(Category::class); }
public function tags() { return $this->belongsToMany(Tag::class); }
public function comments() { return $this->hasMany(Comment::class); }
public function likes() { return $this->hasMany(Like::class); }

// ... diğer modeller de aynı mantıkla
```
